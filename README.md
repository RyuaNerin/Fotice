# Fotice

- Fotice 는 파이널 판타지 14 인게임 채팅방에 공지사항이 올라오는 것을 차단합니다.

## 요구 사항

- Windows 10 환경에서 실행하는 DX11 클라언트.

## 사용방법

- [여기](https://github.com/RyuaNerin/Fotice/releases/latest) 에서 마지막 릴리즈를 **적당한 위치** 에 다운로드 해주세요.

- 다운로드 받은 파일을 실행해주세요.

- 파이널 판타지 14 를 키고 *적용* 버튼을 눌러주세요.
